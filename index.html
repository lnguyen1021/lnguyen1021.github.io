Machine Learning using a Hepatitis Dataset

For this project, we used a dataset from openml.org with twenty features and one million entries to train our model to predict whether an individual who is hepatitis positive is at risk for a life-threatening condition. We were able to train the model using tensorflow, keras, and scikit-learn. After exploring several algorithms, we found a random forest produced the highest accuracy score of 93%. Using tableau desktop, we created a simple tool that allows hepatitis positive users to input several fields and will predict an individuals risk of a life-threatening condition.
